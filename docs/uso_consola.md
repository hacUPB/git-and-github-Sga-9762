# Conceptos aprendidos sobre la utilización de la consola en GitBash
## **¿Cómo se puede utilizar la consola para navegar a través de directorios y carpetas?**
Se pueden utilizar los comandos "cd" y "ls" para navegar mediante directorios locales.

El comando "cd" sirve para "moverse" a través de estas carpetas; por ejemplo, si deseo entrar a la carpeta de documentos, escribo "cd Documents (o el nombre de la carpeta en el sistema)", y, si esta existe, el programa la ubica y se "enfoca" en esta. 

Si deseo ver o listar los archivos y carpetas que se encuentren en "Documentos", utilizo el comando "ls", y la salida arrojará todos los archivos ubicados en esa carpeta.

## **¿Cómo se crean archivos y directorios en la consola?**

Utilizando los comandos "touch" y "mkdir" podemos crear archivos y directorios. 

El comando "touch" se utiliza para crear archivos, como por ejemplo un archivo markdown, o un script de python. La sintaxis es: "touch (nombre.extension)".

El comando "mkdir" sirve para crear directorios. La sintaxis es: "mkdir (nombre carpeta)". 

**Importante:** Antes de crear cualquier directorio o archivo se debe navegar al directorio en los cuales estos archivos serán alojados, utilizando los comandos descritos en el literal anterior.

## **Principales comandos utilizados en esta unidad:**

- *cd*
- *ls*
- *touch*
- *mkdir*
